# Headphone Shop App

A simple e-commerce web application where users can browse headphones, add products to a cart, and manage their selected items.

## Features

* Browse different headphones on the home page
* Add headphones to the shopping cart
* Remove items from the cart
* View all selected items in the cart page
* Simple shopping cart functionality
